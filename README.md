# Student Grade Calculator

A grade calculator I built that computes letter grades, 
GPA per subject and overall CGPA. It also tells you 
whether you made the Dean's List, passed or failed.
Supports multiple students in one session.

## What it does
- Enter any number of subjects and marks
- Calculates letter grade and GPA for each subject
- Calculates overall CGPA
- Shows Dean's List / Pass / Fail status
- Can repeat for another student without restarting

## Grading scale
- A+ / A / A- → 4.00 / 4.00 / 3.70
- B+ / B / B- → 3.30 / 3.00 / 2.70
- C+ / C / C- → 2.30 / 2.00 / 1.70
- D → 1.00 / F → 0.00

## Built with
- Python

## How to run it
python grade_calculator.py
